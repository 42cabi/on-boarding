### 참고사항

- Vite로 React 프로젝트 생성하기

    `$ npm create vite@latest` // react, js 선택

    `cd 본인이 생성한 프로젝트`

    `npm i`

    `npm i styled-components`

- Vite로 React 프로젝트 실행하기

    `npm run dev`

    src 안에 있는 App.jsx에서 작업하기
