# 구현 목록

## 입력부
- [ ] 파싱
  - [ ] `<상품>커맨드`를 파싱하는 기능
  - [ ] <> 내부의 상품을 Product로 생성하는 기능
  - [ ] <>를 제외한 것들을 커맨드로 생성하는 기능
- 기능
  - [ ] 계산 : 길이 * UNIT(1000)
    - 담기
    - [ ] 커맨드 없이 <상품>
    - [ ] <상품>랑
    - [ ] <상품>이랑<상품>
  - [ ] 빼기 : 해당 상품을 제외하는 기능
  - [ ] 찾기 : 해당 상품을 찾는 기능
  - [ ] 개수 세기 : 상품의 개수를 세는 기능
  - [ ] 주문 완료
    - [ ] <상품> 없이 커맨드만 존재
    - [ ] 총 계산값을 반환하는 기능
